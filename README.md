1) What is the difference between null and undefined?
Answer: Null is empty value where programmers intentionally assign it to variable. On the other hand, undefined is where variable is declared but no value is assigned.
2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Answer: Map() takes every value of an array, can modify it and then returns a new array. While forEach do same as Map() but don't return anything.
3) What is the difference between == and ===?
Answer: == is loosely check type of data and only compare value. But === strictly check, it compare both value and data type. 
4) What is the significance of async/await in fetching API data?
Answer: async/await makes asynchronous code look synchronous. It improves readability.
5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Answer: Global - It is declared outside of any function and can be accessed from anywhere. Function - It is declared within function and only accessible within that function, can no access from outside. Block - It is declared inside block between curly bracket and only accessible inside block. Here variables are declared with let or const
